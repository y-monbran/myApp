# README

## アプリ名:  
努力可視化メーター

## 概要:  
毎日の努力を可視化することができるアプリ。

## 制作背景(意図):  
努力は時間で語るものではないと思いますが、少なくとも自分で自分を褒めることは必要だと思っていました。
普段見えない努力した時間を可視化することによって、努力を積み上げた実感を感じたり、モチベーションアップに繋がるようにしたかったので作成しました。
